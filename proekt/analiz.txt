Анализ ниши «МедВычет»
Что есть в проекте
Продукт технически готов на 80%+: OCR-пайплайн (QR + EasyOCR/Tesseract/OpenAI Vision), Telegram-бот, ЛК, экспорт PDF-реестра для ИФНС, 152-ФЗ. Это сильная позиция — конкуренты такой автоматизации по чекам аптек не имеют.

Выводы: рынок платных вычет-сервисов = ~1 млрд ₽/год у двух лидеров, растёт кратно, маржа чистой прибыли ~5% (тянут штат экспертов). Выручка 400 млн при 18 сотрудниках = модель работает. Прямого конкурента по OCR аптечных чеков нет — есть только кэшбэк-приложения («Чеки из Аптеки») и бесплатная «Проверка чеков ФНС» без налоговой логики.

Критический факт — двойной
Хорошо: с 2025 упрощённый вычет (клиника сама шлёт справку в ФНС) покрывает только медуслуги. Для лекарств упрощёнка НЕ работает: по-прежнему нужны 3-НДФЛ + чеки + рецепты (КонсультантПлюс, rnk.ru). Ниша проекта защищена законом — рутину с чеками никто не отменил.

Плохо: глава ФНС публично заявлял курс на перевод всех вычетов в упрощёнку (РБК). Горизонт жизни ниши «только лекарства» — оценочно 3–5 лет. Строить надо шире.

Юнит-экономика клиента — главный риск
Средний возврат только за лекарства: 13% × 15–40 тыс ₽ трат = 2 000–5 200 ₽. За сервис при таком возврате платят максимум 300–700 ₽. Standalone «чеки лекарств» — маленький чек, не бизнес.

Но: лекарства + лечение + стоматология + обучение + фитнес в одной декларации = возврат 10–19,5 тыс ₽ → готовность платить 1500–3000 ₽. Лекарства = крючок и накопительная привычка (чеки копятся весь год), монетизация = полный пакет соцвычета.

Вердикт
Ниша выгодная как клин, невыгодная как изолированный продукт. Вход дешёвый: продукт готов, лидеры доказали выручку 400 млн ₽, прямой OCR-конкуренции нет. Реалистичная цель года 1–2: 5–15 млн ₽ выручки при затратах на маркетинг 1–3 млн ₽.

План развития и цены
Этап 1 (0–6 мес) — запуск, лекарства как магнит:

Free: хранение/распознавание чеков, калькулятор возврата. Цель — привычка «сфоткал чек в бот».
Тариф «Документы»: 499 ₽/год — готовый пакет (реестр PDF + 3-НДФЛ-данные) для самостоятельной подачи. Демпинг против 1690 ₽ НДФЛки, маржа ~100% (всё автоматом, экспертов нет).
Канал: Telegram Ads на ЦА «хронические заболевания, родители, пенсионеры-дети», ~0,5–1 млн ₽ теста.
Этап 2 (6–12 мес) — полный соцвычет:

Тариф «Под ключ»: 1990 ₽ — лечение + лекарства + обучение, проверка экспертом, подача через ЛК ФНС. Прямая конкуренция с НДФЛкой по цене −30%.
Модель «% от возврата» (5–7%, мин. 499 ₽) — психологически легче, платят после денег.
Этап 3 (12–24 мес) — B2B2C, защита от упрощёнки:

White-label аптечным сетям и телемедицине («вернём 13% за лекарства» = их программа лояльности), оплата за активного пользователя 50–150 ₽.
ДМС-страховщики, работодатели (льгота сотрудникам).
Расширение: имущественный вычет, ИИС — там средний чек НДФЛки 3–5 тыс ₽.
Риски: упрощёнка ФНС накроет лекарства (мониторить законопроекты); сезонность (пик январь–апрель — копилка чеков сглаживает); требование рецепта врача отсекает часть чеков (нужен UX-гайд «как взять рецепт»).
